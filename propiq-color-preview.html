If this looks right — one CSS variable change updates everything
The entire UI kit uses --g as the primary color variable. Changing that one variable across all 17 files replaces every green button, sidebar active state, badge, link, and KPI number in the whole platform.
/* Old */
--g: #1a7a56;  /* remove */
--gl: #e8f5ef; /* remove */
--gm: #0f5c3f; /* remove */

/* New */
--g: #1a2744;  /* Deep Slate — primary */
--gl: #e8ecf5; /* Slate light bg */
--gm: #111d33; /* Slate hover */

/* Keep trend/success green for ↑↓ numbers only */
--success: #1a7a56;
