GitHub Pages — Free, Instant, Shareable Link
Put all files in one GitHub repo, enable Pages, and you get a live URL like graehamwatts.github.io/propiq-ui that anyone can open. The entire kit becomes shareable in one link — dev team, investors, brokers, anyone.
# 1. Download all files from Claude as a zip
# 2. Create a new repo at github.com — name it "propiq-ui"
# 3. Upload all .html files (drag & drop in GitHub UI)
# 4. Settings → Pages → Source: main branch → Save
# 5. Your URL: https://[username].github.io/propiq-ui/propiq-ui-index.html
Total file size is under 3MB — well within GitHub's free limits. No server, no cost, no expiry. Files stay in your repo permanently.
